#  RatesConverter

**RatesConverter** — это десктопное приложение с графическим интерфейсом для удобной конвертации валют. Проект написан на Python с использованием фреймворка PyQt6 и библиотеки Pillow для динамической обработки изображений. База курсов хранится локально, что обеспечивает полную автономность работы приложения.

## Требования
- Python 3.10 или новее.
- Git (для клонирования репозитория).

## Установка и запуск

Следуйте этим шагам, чтобы развернуть проект на своем компьютере:

**1. Склонируйте репозиторий:**
```git clone https://github.com/Gaznmazn/RatesConverter ```

**2. Перейдите в папку с проектом:**
```cd RatesConverter```

**3. Создайте виртуальное окружение:**
```python -m venv venv```

**4. Активируйте виртуальное окружение:**
* Для Windows (PowerShell):
  ```.\venv\Scripts\activate```
  *(Если возникает красная ошибка безопасности, сначала выполните команду `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`, а затем повторите попытку активации).*
* Для macOS/Linux:
  ```source venv/bin/activate```

**5. Установите необходимые зависимости:**
```pip install -r requirements.txt```

**6. Запустите приложение:**
```python main.py```

##  Горячие клавиши
В приложении предусмотрено управление с клавиатуры для быстрого доступа к основным функциям:

* **`Ctrl + R`** — быстрая смена выбранных валют местами (Reverse).
* **`Ctrl + T`** — мгновенное переключение интерфейса между светлой и темной темой оформления.
