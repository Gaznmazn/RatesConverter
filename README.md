# Конвертер валют (PyQt6)

Десктопное приложение для кросс-конвертации валют с динамической подгрузкой флагов, разработанное в рамках учебной вычислительной практики.

## Установка и запуск
1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python main.py`
